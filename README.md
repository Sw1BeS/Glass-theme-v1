## 🚀 Quick Start

### Prerequisites
- Shopify Partner Account
- Shopify CLI v3+
- Node.js 16+

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sw1BeS/Glass-theme-v1.git
   cd Glass-theme-v1
   ```

2. **Connect to your store**
   ```bash
   shopify theme dev
   ```

3. **Deploy to production**
   ```bash
   shopify theme push
   ```
